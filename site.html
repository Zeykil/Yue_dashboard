Sobre o Yue

Yue é um bot de Discord em desenvolvimento, criado por um único desenvolvedor apaixonado por trazer funcionalidades úteis e divertidas para comunidades. Ainda em fase inicial, o Yue já conta com comandos essenciais para moderação e interação.

O objetivo é criar um bot leve, eficiente e fácil de usar, com foco em melhorar a experiência dos usuários no Discord. Yue é perfeito para servidores que buscam uma solução simples e personalizável.

Comandos Principais

/comandos

Lista todos os comandos disponíveis no Yue. Use este comando para descobrir tudo o que o bot pode fazer.

/sobre

Mostra informações sobre o bot, como versão, desenvolvedor e links importantes.

/clear [número]

Limpa um número específico de mensagens no chat. Ideal para manter o servidor organizado.

/ajuda

Exibe uma mensagem de ajuda com instruções sobre como usar o Yue.

Recursos em Desenvolvimento

Moderação

Ferramentas para banir, expulsar e gerenciar mensagens indesejadas. Mantenha seu servidor seguro e organizado.

Interação

Comandos divertidos para engajar a comunidade, como respostas automáticas e mensagens personalizadas.

Personalização

Permite que administradores configurem o bot de acordo com as necessidades do servidor.

Estatísticas

Obtenha dados detalhados sobre a atividade do servidor, como número de mensagens e usuários ativos.

Desenvolvimento

Yue está em constante desenvolvimento, com novas funcionalidades sendo adicionadas regularmente. Como um projeto de um único desenvolvedor, o progresso pode ser lento, mas cada atualização é feita com cuidado e dedicação.

Se você tem alguma sugestão ou deseja contribuir, sinta-se à vontade para entrar em contato! O código-fonte está disponível no GitHub, e estamos abertos a colaborações.
